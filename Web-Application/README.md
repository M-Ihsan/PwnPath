# Web Application Security

## Overview
Web application attacks target the most exposed 
part of any organisation — their websites and 
web apps. Directly facing the internet, accessible 
by anyone in the world.

## Lab Environment
- Attacker: Parrot OS + Burp Suite Pro
- Target: DVWA (Damn Vulnerable Web Application)
- Additional: OWASP Juice Shop, PortSwigger Labs
- Network: Local + VMware

## Attacker Methodology
Before attacking, always follow this order:

1 - Recon — what pages exist, what inputs exist
2 - Map all inputs — forms, URLs, headers, cookies
3 - Test each input systematically
4 - Document everything as you go
5 - Write findings in report format
